Learning basics of Node.js through Pluralsight examples

To run the app
```bash
npm install bower -g
```
```bash
npm install
```
```bash
bower install
```
Run mongodb server e.g
```bash
mongod.exe --dbpath yourdatabasepath
```
Run the server
```bash
node server.js
```
Enter in browser
```bash
http://localhost:3000
```
